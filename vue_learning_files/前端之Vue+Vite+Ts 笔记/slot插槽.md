## 具名插槽
### 只需要在父组件中`<category>`后面写上 v-slot:XX 之后在category`<slot>`后面写上name="XX"即可
但是如果不想一个组件就按一个插槽呢？想分插槽？
只需要在`<category>`下面写多个`<template>`之后在每个template后面写v-slot:XX就可以实现
##### v-slot可以用\#代替！
## 作用域插槽（压岁钱在子，买东西由父定）
信息在子组件时，父组件使用上述插槽调用会出问题
这时需要使用作用域插槽：
首先，信息定义在子组件的ts部分，假设名为X
子组件html写作用域插槽
```html
<slot :y="x"></slot>
```
父组件要调用子组件信息时
```html
<子>
<template v-slot="y">
</template>
</子>
```
