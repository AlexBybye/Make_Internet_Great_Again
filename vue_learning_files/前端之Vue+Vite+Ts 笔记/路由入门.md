## 开始创建路由（index.ts）
```ts
//创建路由器并暴露
import { createRouter,createWebHistory} from 'vue-router'
import Home from '@/component/Home.vue'
//create router
const router = createRouter({
    history:createWebHistory(),
    routes:[
        {
            name:'yeah',
            path:'/home',
            component:Home,
        },
    ]
})
//export router
export default router
```
### 注意：上述为固定代码，先引入工具；再创建具体：path规定的是网页地址，component：组件名； 最后再export就可以
### 此外，一般组件习惯于放在component的下面

## 然后去搞母路由页面app.vue
```vue
<template>
  <div class="app">
    <h1 class="title">Vue路由测试</h1>
    <!-- 导航区 -->
    <div class="navigate">
      <RouterLink to="/home" active-class="xiaozhupeiqi">首页</RouterLink>
    <!-- RounterLink就是个组件-->
    </div>
    <!-- 展示区 -->
    <div class="main-content">
      <RouterView></RouterView>
    </div>
  </div>
</template>
<script lang='ts' setup name="App">
import { RouterView } from 'vue-router'
</script>
<style>
.navigate a {
  display: block;
  text-align: center;
  width: 90px;

  border-radius: 10px;

  text-decoration: none;
  color: white;

}
.navigate a.xiaozhupeiqi {
  background-color: #64967E;

}
</style>
```
## 这个文件哪里变动了呢？
1：哪里有进入子路由的链接，哪里
`<RouterLink to="/home" active-class="xiaozhupeiqi">首页</RouterLink>` 
`<RouterLink to ="path内容" active-class="">`

2：这里CSS很有意思，
 因为这里是在div navigator下面，所以要先指定navigator CSS对象名字，
 之后.navigator 对象名字.控制CSS的名字这样去做CSS样式头

3：注意要搞一个展示区，然后里边写上`<RouterView>` 在script中也要引入RV！

## 最后去搞位于外层的main.ts去引入应用及其路由器
```ts
import { createApp } from 'vue'//引入来创建应用
import App from './App.vue'//引入根组件
import router from './router'//引入路由器
const app = createApp(App)//创建应用
app.use(router)//使用路由器
app.mount('#app')//挂载应用
```
