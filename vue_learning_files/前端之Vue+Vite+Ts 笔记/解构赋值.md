## 数组解构
```javascript
const arr = [1, 2, 3];
const [a, b, c] = arr;
console.log(a); // 1
```
## 对象解构
```javascript
const obj = { name: "Vue", version: 3 };
const { name, version } = obj;
console.log(name); // "Vue"
console.log(version); // 3
```
